Pharoscion Global Solutions, Pune, India | June 2022 – October 2023

I worked as a Software QA Tester on the company’s flagship Internet Banking System project, a highly critical platform used by banking customers for online transactions, cheque management, fixed deposit (FD) handling, account monitoring, and financial reporting. Given the sensitive nature of banking applications, my role required a strong focus on accuracy, security, usability, and compliance with financial industry standards.

This experience allowed me to work in a fast-paced agile environment where quality assurance was not just about detecting defects, but also about contributing to a seamless, secure, and user-friendly banking experience.

Project Context & Scope

  • Ensuring data integrity and accuracy across financial modules.
  • Maintaining a consistent experience across browsers and devices.
  • Supporting frequent releases in short sprint cycles without compromising stability.
  • Meeting audit and compliance requirements demanded by financial regulators.

As a QA tester, I was deeply involved from requirement analysis through release sign-off, ensuring full coverage and traceability of test cases.

Core Responsibilities

1. Test Design & Execution

  • Translated functional and business requirements into detailed test cases.
  • Created manual test scenarios for functional, regression, smoke, and integration testing.
  • Maintained a requirement traceability matrix (RTM) using Excel to ensure test coverage.
  • Conducted data-driven testing for transaction workflows (different account types, currencies, transaction limits, etc.).

2. Defect Tracking & Quality Metrics

  • Logged issues into JIRA, categorized by severity and priority.
  • Conducted root cause analysis discussions with developers to identify recurring issues.
  • Maintained a defect density chart for each release cycle to measure product stability.
  • Ensured defects were resolved within the sprint and re-tested in subsequent builds.

3. Test Automation Development

I designed and maintained an automation framework tailored for banking workflows:

Tools/Tech Stack: Selenium WebDriver, Java, TestNG, Maven, Jenkins.

  • Page Object Model (POM) for better script maintainability.
  • Data-driven testing using TestNG annotations and external test data files.
  • Parallel test execution to speed up regression runs.
  • Customized ExtentReports integration for execution logs, screenshots, and audit-ready HTML reports.

Automated workflows included:

  • User login & session management.
  • Fund transfers and transaction confirmation.
  • FD creation, maturity, and closure.
  • Cheque request and status validation.

The automation framework reduced manual regression effort by ~40%, ensuring quicker release validation.

4. API & Backend Validation

  • Performed REST API testing using Postman.
  • Validated APIs for account balances, transaction history, FD creation, and cheque status.
  • Ensured frontend and backend data consistency by comparing API responses with UI outputs.
  • Designed positive, negative, and boundary test cases for API endpoints.

5. Cross-Browser & Mobile Testing

  • Tested the application across Chrome, Firefox, Edge, and Safari.
  • Performed responsive testing for mobile devices to ensure compatibility with Android and iOS browsers.
  • Verified UI/UX consistency and usability across platforms.

6. Agile Process & Collaboration

  • Participated in daily stand-ups, sprint planning, and retrospective meetings.
  • Worked closely with developers to clarify requirements and reproduce bugs.
  • Supported User Acceptance Testing (UAT) with clients, guiding them through test execution and documenting feedback.
  • Contributed to test planning, sprint reports, and release notes for stakeholders.

7. CI/CD Integration

  • Configured Jenkins pipelines to trigger test automation runs after each code commit.
  • Implemented nightly regression test runs with automated reporting.
  • Ensured quick feedback loops for developers, supporting continuous integration practices.

Key Achievements

  • Built a reusable automation framework that accelerated regression testing and reduced human error.
  • Delivered audit-ready HTML reports with screenshots and detailed logs, supporting financial compliance.
  • Successfully executed cross-platform testing ensuring seamless banking services on web and mobile.
  • Improved defect detection and resolution rate by maintaining strong communication with developers.
  • Contributed to front-end development fixes, gaining additional hands-on experience with web technologies.
  • Enhanced overall product quality, stability, and customer satisfaction by ensuring bug-free releases.

Skills & Tools Gained

  • Testing Tools & Frameworks: Selenium WebDriver, TestNG, Maven, Jenkins, Postman.
  • Defect Tracking & Reporting: JIRA, ExtentReports, Excel-based RTM.
  • Testing Types: Functional, Regression, Integration, Smoke, System, UAT.
  • Development & CI/CD: Agile methodology, front-end debugging, Jenkins pipelines.
  • Soft Skills: Team collaboration, client communication, agile reporting, documentation.

Overall Impact

My work at Pharoscion Global Solutions was crucial in ensuring the stability, reliability, and compliance of a mission-critical Internet Banking System. By combining manual and automated testing approaches, I helped deliver a robust platform that customers could trust for handling their financial transactions.

This role not only strengthened my expertise in Selenium automation and API testing, but also gave me deep insights into agile product delivery, banking domain workflows, and CI/CD practices—skills that continue to shape my career as a QA and automation professional.
